## Installation

```bash
npm install
```

## Create Module

```bash
nest g module {name}
```

## Create Controller

```bash
nest g controller {name} --no-spec
```

## Create Service

```bash
nest g service {name} --no-spec
```

## Installation Mongoose

```bash
npm i mongoose @nestjs/mongoose @nestjs/config --save
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
